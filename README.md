# Borders

Um projeto de inglês inspirado na música **“Borders”**, da M.I.A., desenvolvido com foco em apresentação visual, organização de conteúdo e navegação simples entre diferentes tópicos do trabalho.

## Sobre o Projeto

Esse projeto foi criado para explorar o tema da música **Borders**, relacionando letra, contexto histórico, atualidades e o uso de inteligência artificial no processo de construção do site.

A proposta principal foi montar uma apresentação em formato de site, com linguagem direta e apoio visual, para facilitar a exposição oral e tornar o conteúdo mais dinâmico e interessante.

O projeto contém versões curtas e objetivas das informações, além de páginas específicas para aprofundar cada parte da atividade.

## Funcionalidades

- Página inicial com apresentação do tema
- Seção sobre a artista M.I.A.
- Análise da letra da música
- Contexto sobre migração e refugiados
- Página com notícias e atualidades relacionadas ao tema
- Página explicando o uso de IA no projeto
- Navegação entre páginas
- Interface responsiva e organizada

## Tecnologias Utilizadas

- HTML
- CSS
- JavaScript

## Objetivos do Projeto

- Apresentar o tema da música de forma clara e visual
- Relacionar a letra com migração e direitos humanos
- Organizar o conteúdo em seções separadas
- Praticar desenvolvimento front-end
- Criar uma apresentação mais moderna para o trabalho escolar
- Explorar o uso de IA de forma consciente e revisada
